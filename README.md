# PY MATH LABS

A repository containing mathematical examples and algorithms in Python, 
focused on small educational implementations (e.g., Brun’s constant).

![logo](static/logo.svg)

***

## Project Structure
```
python-math-is-fun/
├── algorithms/                                 # Algorithms implementations
│   └── 01_easter.py
├── math/                                       # Mathematical implementations
│   ├── 01_nepero.py
│   ├── 02_brun_constant_twins_numbers.py
├── requirements.txt                            # Dependencies
└── README.md                 
```

***

## Requirements

1. **Python 3.8+**
2. Install dependencies (if any):
   ```bash
   python -m pip install -r requirements.txt
   ```

***

## How to run

You can execute each example or math file directly using Python.  
For example:

```bash
python algorithms/01_easter.py
```

or

```bash
python math/02_brun_constant_twins_numbers.py
```
